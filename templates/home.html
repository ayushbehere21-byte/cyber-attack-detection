AI Cyber Attack Detection & Monitoring System

Real-Time Cyber Threat Detection Platform

Cyber attacks such as brute force login attempts and phishing websites are becoming increasingly common. This system detects suspicious login activity, identifies phishing links, and records attacker IP addresses along with their geographic location. The platform provides an administrative dashboard that helps security teams monitor login attempts, detect cyber attacks, and analyze attacker behavior in real time.

Key Features

Brute Force Detection

Detects repeated login failures and possible hacking attempts.

Phishing Detection

Analyzes URLs and identifies suspicious phishing websites.

Threat Monitoring

Displays login attempts and attacker IP logs.
